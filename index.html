Webhook Amp

Example URL: 'https://2rygsn9pzc.execute-api.us-east-1.amazonaws.com/dev/amp?url=http://www.google.com&url=http://bing.com'

This servers allows you to send the same singular webhook request to multiple endpoints

This service accepts multiple valid URLS with a key of 'url' in the query string to build the list of endpoints to forward the request too. Url Parser uses this standard so please ensure your urls are compliant. Example query string would look like /amp?url=http://www.google.com&url=http://bing.com You can use this endpoint as an example: 'https://2rygsn9pzc.execute-api.us-east-1.amazonaws.com/dev/amp' Please note that all your webhook data will pass through this service, so if security or confidentiality are important I suggest you clone the Repo and deploy yourself using serverless deploy
